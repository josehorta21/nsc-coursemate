# NSC CourseMate

A course planning platform for North Seattle College students. Plan quarters, track prerequisites, monitor degree progress, and visualize your path to graduation.

## Status
In active development — AD350 Spring 2026 project (Week 05)

## Tech Stack
- **Database:** PostgreSQL hosted on Supabase
- **Backend:** Supabase REST API + Row Level Security
- **Frontend:** React + Vite + TypeScript + Tailwind CSS
- **Testing:** Postman
- **Deployment:** Vercel + Supabase

## Features (Planned)
- Student profiles with program enrollment
- Course catalog with prerequisites tree
- Quarter-by-quarter academic planner
- Degree progress tracker with credit calculations
- Prerequisite validation engine

## Project Structure
- db/ — SQL schema, seed data, ER diagrams
- docs/ — Technical documentation, API reference
